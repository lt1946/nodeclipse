Nodeclipse is an plugin that adds IDE functionality to the Eclipse platform for the Node.js.

The purpose of Nodeclipse is to create an environment in which development of Node.js is easy for a beginner to intermediate user. Development is slow, but ongoing. If you would like to help out, please leave a comment stating so in the Nodeclipse forum.

Features
